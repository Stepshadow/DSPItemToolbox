\# DSP Item Toolbox



戴森球计划物品工具箱 Mod。



\## 功能



\- 按 `=` 打开或关闭工具箱

\- 搜索游戏物品和建筑

\- 输入数量后点击物品即可加入背包

\- 显示游戏内物品图标

\- 工具箱打开时锁定地图镜头和游戏输入



\## 开发环境



\- Dyson Sphere Program

\- BepInEx 5

\- .NET Framework 4.7.2

\- Harmony



\## 安装



将编译后的 `DSPItemToolbox.dll` 放进：
